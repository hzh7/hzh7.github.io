unity3d开发总结（二）– 玩家脚本解读

Snake脚本是游戏的核心脚本之一，主要用于控制玩家的行为，包括蛇的运动，加速，吃食物，身体变长，击杀AI，死亡等一系列行为。

初始化

InvokeRepeating("Move", 0f, Time.deltaTime );

这个函数表示开始执行后的0秒后调用Move()函数，并以每帧的时间间隔重复执行该函数。其实这里将Move()中的内容放在Update()函数中也行，不过为了增加代码可读性我这里写成了一个独立的函数。另外与InvokePepeating()函数类似的函数还有Invoke()函数，都是unity引擎提供的函数，具体用法可查阅官方api文档。

初始化中做的另外一件事是生成我们的玩家蛇。

for (int i = 0; i < len; i++)
{
    Vector2 p = new Vector2(myTransform.position.x-(i+1)*1f, myTransform.position.y);
    GameObject g = (GameObject)Instantiate(PlayerTailPrefab, p, Quaternion.identity);
    tail.Insert(i, g.transform);
}
for (int i = 0; i < len * disnub + 1; i++)
{
    Vector2 p = new Vector2(myTransform.position.x - i * 1 / (float)disnub, myTransform.position.y);
    posi.Add(p);
}

其中tail列表中放的是蛇的身体，posi列表中初始化存放了蛇身的之后运动的位置。之后的运动将用这个列表记录蛇头每帧的运动位置，以蛇头经过的位置的坐标给之后的蛇身。

Update函数

该函数中主要作用有：

  • 作蛇的转向，先得到蛇头当前的朝向，若当前的方向和目标的方向不一致时则每帧进行转动使得蛇的方向为我们的目标方向。

    获得当前的方向，即z的值：

    Angle = myTransform.localEulerAngles.z;

    转向：

    targetRotation = Quaternion.Euler(0f, 0f, (float)tarAngle);
myTransform.rotation = Quaternion.Slerp(myTransform.rotation, targetRotation, Time.deltaTime *5);

  • 统计当前分数

    score = tail.Count * 6 + eatsmafdcnt;

  • 蛇身变大

    myTransform.localScale = new Vector3(tail.Count * 0.001f + 0.5f, tail.Count * 0.001f + 0.5f, 1f);
for (int i = 0; i < tail.Count; i++)
{
    tail[i].transform.localScale = myTransform.localScale;
}

    这几句代码实现的功能是当蛇的长度越来越长的时候将蛇按比例进行放大。

Move函数

void Move()
{
    myTransform.Translate(dir * Time.deltaTime * 5);
    if (ate)
    {
        posi.Insert(0, myTransform.position);
        mycount--;
        if (mycount == 0)
        {
            mp = posi[posi.Count - 1];
            mg = (GameObject)Instantiate(PlayerTailPrefab, mp, Quaternion.identity);
            tail.Insert(tail.Count - 1, mg.transform);
            ate = false;
            mycount = disnub;
        }
        for (int i = 0; i < tail.Count; i++)
        {
            tail[i].position = posi[(i + 1) * disnub];
        }
    }
    else if (tail.Count > 0)
    {
        posi.Insert(0, myTransform.position);
        for (int i = 0; i < tail.Count; i++)
        {
            tail[i].position = posi[(i + 1) * disnub];
        }
        posi.RemoveAt(posi.Count - 1);
    }
}

Move函数主要是记录蛇头经过的点的坐标，并把身体的坐标进行变化使得蛇整体在前进。其中又分为蛇是否增加身体，当蛇没有增加身体长度时直接记录蛇头的位置并去posi中的相应点给身体，若蛇有吃到食物并增加身体时，需要往tail中插入（好像我这里用的tail.Insert(..)=tail.add()因为是之前边学边写的之后就没有改），不用将多的posi的数据移除。

加速并消耗身体

public void ButtonControlPressed()
{
    if (isLive && score > 24)
    {
        if (yanshi > 0)
        {
            yanshi--;
        }
        else if (eatsmafdcnt > 0)
        {
            eatsmafdcnt--;
            yanshi = 30;
            SpawnFood.Instance.Spawn(tail[tail.Count - 1].transform.position);
        }
        else
        {
            Destroy(tail[tail.Count - 1].gameObject);
            tail.Remove(tail[tail.Count-1]);
            eatsmafdcnt = 6;
        }

        Move();
    }
}

其实这个功能的实现并不难，当按下按钮时，会执行上面这个函数，函数的内容是先判断蛇是活的并且长度（即分数）大于24，此时满足可以加速的条件，则在这里多执行一次Move（）函数。所谓的身体消耗就是加速的时候会在蛇尾拉出食物并减少分数，玩过游戏的朋友应该都清楚，这里为了不让分数减得过快，我用了一个很笨的办法，就是另设一个参数自减到0达到延时的效果。

判断死亡

在脚本中用一个全局变量isLive来表示蛇的存活状态，当蛇因为撞墙或者其他蛇死掉时会将这个变量置为false，并调用Live（）函数，在这个函数中将蛇身体对应的对象进行销毁并在相应位置生成大食物，然后让游戏结束的界面显示出来。值得注意的是蛇头并没有销毁而是让它停止运动（CancelInvoke(“Move”)）并不可见（transform.gameObject.SetActive(false)）。这样做的目的是因为游戏结束界面需要显示的分数和击杀是根据玩家控制的蛇的获取的，当蛇头也进行销毁时会导致这些数据的消失。虽然我一直觉得这是一种比较蠢的方法，但初学者的我想着先实现后有能力再去优化吧。

尾话

由于时间过去比较久了当时一些代码实现的想法记得并不是很清楚了所以这篇文章的表述有些地方可能不清楚，然后又因为偷懒这篇文章也是拖了写了好几天了。当初我在游戏开发中遇到的难点问题是蛇的移动实现。当初遇到的问题是蛇的移动不够平滑，感觉很卡，因为是模仿像素的贪吃蛇的移动方式。尝试过很多办法，最后通过记录蛇头移动位置，用list去实现一个队列的功能，把这个队列中相应值给对应身体作为坐标。

public static Snake InstanceSnake;
Transform myTransform;
double Angle = 0f;              //当前角度
double tarAngle = 0f;           //目标角度
private Vector2 dir = Vector2.right;    //移动的前进方向
List<Transform> tail = new List<Transform>();   
List<Transform> Body = new List<Transform>();
List<Vector2> posi = new List<Vector2>();   //用于存放蛇头经过的路径位置
int score = 0;                  //分数（蛇的长度）
int killnub = 0;                //击杀蛇的数量
bool ate = false;               //迟到了食物
public bool isLive = true;             //表示存活
public GameObject PlayerTailPrefab;
public GameObject BodyPrefab;
int disnub = 10;                //一节身体移动到前一节的步数
int mycount = 10;               //记录吃掉食物后5帧在长尾巴
int eatsmafdcnt = 0;            //每吃6个小点长一节尾巴
int yanshi = 30;                //用于加速延迟拉屎
bool enChuan = true;            //可进入传送门
Quaternion targetRotation;      //蛇头的旋转

//move里用到的变量，因会反复调用故声明为全局变量、
Vector2 mv;
Vector2 mp;
GameObject mg;

void Awake()
{
    if (InstanceSnake != null)
    {
        Debug.LogError("Multiple instances of SpecialEffectsHelper!");
    }
    InstanceSnake = this;
    myTransform = transform;
}
void Start () {
    InvokeRepeating("Move", 0f, Time.deltaTime );
    int len = 4;
    for (int i = 0; i < len; i++)
    {
        Vector2 p = new Vector2(myTransform.position.x-(i+1)*1f, myTransform.position.y);
        GameObject g = (GameObject)Instantiate(PlayerTailPrefab, p, Quaternion.identity);
        tail.Insert(i, g.transform);
    }
    for (int i = 0; i < len * disnub + 1; i++)
    {
        Vector2 p = new Vector2(myTransform.position.x - i * 1 / (float)disnub, myTransform.position.y);
        posi.Add(p);
    }
    //Collider2D[] hitColliders = Physics2D.OverlapCircleAll(transform.position, 0.5f);
}
void Update () {
    Angle = myTransform.localEulerAngles.z;
    Angle = (Angle > 180 ? Angle - 360 : Angle);
    if (Angle != tarAngle)
    {
        targetRotation = Quaternion.Euler(0f, 0f, (float)tarAngle);
        myTransform.rotation = Quaternion.Slerp(myTransform.rotation, targetRotation, Time.deltaTime *5);
    }
    score = tail.Count * 6 + eatsmafdcnt;
    myTransform.localScale = new Vector3(tail.Count * 0.001f + 0.5f, tail.Count * 0.001f + 0.5f, 1f);
    for (int i = 0; i < tail.Count; i++)
    {
        tail[i].transform.localScale = myTransform.localScale;
    }
}

public int getScore()
{
    return score;
}

public int getKillnub()
{
    return killnub;
}

void Move()
{
    myTransform.Translate(dir * Time.deltaTime * 5);
    if (ate)
    {
        posi.Insert(0, myTransform.position);
        mycount--;
        if (mycount == 0)
        {
            mp = posi[posi.Count - 1];
            mg = (GameObject)Instantiate(PlayerTailPrefab, mp, Quaternion.identity);
            tail.Insert(tail.Count - 1, mg.transform);
            ate = false;
            mycount = disnub;
        }
        for (int i = 0; i < tail.Count; i++)
        {
            tail[i].position = posi[(i + 1) * disnub];
        }
    }
    else if (tail.Count > 0)
    {
        posi.Insert(0, myTransform.position);
        for (int i = 0; i < tail.Count; i++)
        {
            tail[i].position = posi[(i + 1) * disnub];
        }
        posi.RemoveAt(posi.Count - 1);
    }
}

public void KillSnake()
{
    killnub++;
}

public void JoyStickControlMove(Vector2 direction)
{
    float inputX = direction.x;
    float inputY = direction.y;
    double xy = System.Math.Sqrt(inputX * inputX + inputY * inputY);
    tarAngle = Mathf.Acos(inputX / (float)xy);
    tarAngle = (inputY > 0 ? tarAngle : -tarAngle) / Mathf.PI * 180f;
}

public void ButtonControlPressed()
{
    if (isLive && score > 24)
    {
        if (yanshi > 0)
        {
            yanshi--;
        }
        else if (eatsmafdcnt > 0)
        {
            eatsmafdcnt--;
            yanshi = 30;
            SpawnFood.Instance.Spawn(tail[tail.Count - 1].transform.position);
        }
        else
        {
            Destroy(tail[tail.Count - 1].gameObject);
            tail.Remove(tail[tail.Count-1]);
            eatsmafdcnt = 6;
        }

        Move();
    }
}

public void Live()
{
    if (!isLive)
    {
        Leaderboard.has_limit = false;
        GameObject.Find("script").GetComponent<UDPsocket>().Main();
        CancelInvoke("Move");
        dir = Vector2.zero; 
        transform.gameObject.SetActive(false);
        for (int i = 0; i < tail.Count; i++) {
            Destroy(tail[i].gameObject);
        }

        for (int i = 0; i < tail.Count; i++)
        {
            Vector2 v = tail[i].position;
            v.x += Random.Range(-0.5f, 0.5f);
            v.y += Random.Range(-0.5f, 0.5f);
            GameObject g = SpawnBody(v);
            Body.Insert(0, g.transform);
        }
        var gameOver = FindObjectOfType<GameOver>();
        gameOver.ShowButtons();
        gameOver.ShowImage();
    }
}

void OnTriggerEnter2D(Collider2D coll)
{
    if (coll.name.StartsWith("FoodPrefab"))
    {
        eatsmafdcnt++;
        Destroy(coll.gameObject);
        SpawnFood.Instance.setFoodCount(coll.gameObject.transform);
        if (eatsmafdcnt == 6)
        {
            ate = true;
            eatsmafdcnt = 0;
        }
    }
    if (coll.name.StartsWith("BodyPrefab"))
    {
        Destroy(coll.gameObject);
        ate = true;
    }
    if (coll.name.StartsWith("borderTop") || coll.name.StartsWith("borderBottom") ||
        coll.name.StartsWith("borderLeft") || coll.name.StartsWith("borderRight")) //|| coll.name.StartsWith("TailPrefab(Clone)")
    {
        isLive = false;
        Live();
    }
    if (coll.name.StartsWith("chuansongmenPrefab"))
    {
        if (enChuan)
        {
            enChuan = false;
            Vector2 v = GameObject.Find("script").GetComponent<CsDoor>().ChuanSong(coll);
            transform.position = v;
        }
        else
        {
            enChuan = true;
        }
    }
    if (coll.name.StartsWith("TailPrefab"))
    {
        bool flag = false;
        for (int i = 0; i < tail.Count; i++)
        {
            if (coll.transform == tail[i])
            {
                flag = true;
                break;
            }
        }
        isLive = flag;
        Live();
    }
}

{
   GameObject Gobj = (GameObject)Instantiate(BodyPrefab, v,
                Quaternion.identity); 
    //Gobj.GetComponent<Renderer>().material.color = Color.green;
    return Gobj;
}

unity3d开发总结（一）

因为学校有个游戏开发比赛的机缘，接触到unity并学习了一段时间。从16年的10月底到12月中旬左右，历时不到两个月左右时间，从最开始的一无所知到到最后成功开发出一款简陋的游戏，这之间对于通过unity做游戏开发有了一定的认识。由于一开始就是打算做一款仿贪吃蛇大作战的游戏，是一种2d游戏所以最初是打算用cocosx2d做的，学了一点皮毛之后才了解到unity3d，网上很多说法是unity3d将是游戏开发的主流，并且也更加容易简单上手，并且从网上了解到unity3d从5.0之后的版本后对于2d游戏的开发也有了很好的支持，所以后来决定投入unity怀抱学习。总的来说，unity3d有着更加友好的界面，更加强大的引擎，对于编写代码和程序的调试也更加方便，新人上手还是比较容易的。

游戏说明

这次比赛是让仿做一款小游戏，我们小组选的题目是贪吃蛇大作战。作品效果如下：


游戏主要实现的功能有：

菜单界面的模式选择

可进入无尽模式和限时模式，当选择限时模式的时候会有5分钟的计时，当计时时间到会自动结束游戏。

角色的控制

摇杆控制方向和按钮实现加速，在加速中会伴有消耗长度

屏幕显示

屏幕左上角有记录显示当前的长度和杀击数，右上角有动态显示排行榜，排行榜包括名次、角色ID和长度

传送门（小创新）

场景中会随机出现若干传送门，当角色碰到传送门后会随机出现到另外的传送门的地方

背景音乐

游戏开始时循环播放一段背景音乐，当玩家死亡的时候停止播放

结束界面

当玩家撞墙或者装其他蛇上时会死亡并显示游戏结束的界面，界面有显示长度和杀击数，并可选择返回菜单界面和重新开始。（炫耀，用于分享没有实现）

游戏逻辑

  • 游戏开始后，数据初始化，随机生成500个食物点和20条AI蛇。
  • 蛇自行运动，并每10帧进行方向判断，即是否会撞墙或者撞到其他蛇，若会则避开，若都不会则随机转一定角度，也可能不转。
  • 资源的补充：当有AI蛇死掉或食物被消耗到少于预设值会随机再生成进行资源的补充。
  • 蛇死掉尸体会在原地附件转变为大食物点，蛇每吃6个小食物点会增长一节身体，每吃一个大食物增长一节身体，长度以一个小食物为一分计。
  • 蛇的死亡条件依靠碰撞，吃食物和传送门等也依靠碰撞，这都依赖于unity提供的碰撞检测
  • 显示脚本实时统计场上分数和玩家战绩，并显示与屏幕中

源码

游戏是使用unity3d5的版本开发的，源码现托管于github上：https://github.com/hzh7/gameAssets 之后再将对游戏中的难点做分析并解读代码。
